# ✨ Particle Gesture Controller

An interactive web application that combines **Three.js** and **MediaPipe Hands** to create a real-time 3D particle visualization controlled by hand gestures through a webcam.

## 🚀 Features

- 🖐️ Real-time hand gesture tracking using MediaPipe Hands
- 🌌 Interactive 3D particle system powered by Three.js
- ❤️ Multiple particle templates:
  - Heart
  - Flower
  - Saturn
  - Buddha
  - Fireworks
- 🎨 Live particle color customization
- 📹 Webcam preview with tracking status
- 📊 Hand tension indicators
- 🔄 Smooth particle animations and transitions
- 📱 Fully responsive design
- ✨ Modern glassmorphism user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Three.js
- MediaPipe Hands
- OrbitControls

## ▶️ Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/your-DharshiniK90
   /Particle-Gesture-Controller.git
   ```

2. Open the project folder.

3. Run the project using a local server (such as Live Server in VS Code).

4. Allow webcam access when prompted.

5. Interact with the 3D particle system using your hand gestures.

## 🎮 How to Use

- Grant camera permission.
- Move your hand in front of the webcam.
- Choose a particle template from the control panel.
- Change particle colors using the color picker.
- Enjoy real-time interactive particle animations.
